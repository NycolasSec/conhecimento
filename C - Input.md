```c
#include <stdio.h>
int main(void){

    int value;
    scanf("%d", &value);
    printf("O valor escrito foi %d", value);
}
```
 O método ``scanf`` indica que queremos ler um valor do teclado do usuário. O `%d` indica que será um decimal, e depois da vírgula indicamos a variável que receberá esse valor.
